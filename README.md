# Asketch Portfolio

A one-page portfolio website for Ariel Casia, featuring a dark neon theme, gallery with lightbox, and contact form.

## Features
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **Gallery**: Grid layout with lightbox, zoom/pan support, and smooth transitions.
- **Contact Form**: Integrated with Google Forms for submissions.
- **Theme**: Dark neon aesthetic inspired by Vyro.ai.

## Setup
1. Open `index.html` in your browser.
2. For development, run a local server (e.g., `python -m http.server`).

## Deployment
Ready to deploy on Vercel or Netlify.
